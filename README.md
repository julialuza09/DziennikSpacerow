# Dziennik spacerów

## Opis aplikacji
Aplikacja mobilna umożliwiająca rejestrowanie spacerów użytkownika.
Podczas spaceru zbierane są dane o czasie, dystansie, liczbie kroków
oraz zdjęcia wykonywane w trakcie aktywności.

## Wykorzystane sensory i źródła danych
- GPS – lokalizacja, dystans, prędkość
- Sensor kroków (krokomierz)
- Aparat fotograficzny
- Pamięć urządzenia – zapis danych i zdjęć

## Funkcjonalności
- START / STOP spaceru
- Pomiar czasu, dystansu i kroków
- Dodawanie zdjęć podczas spaceru
- Historia spacerów
- Widok szczegółów spaceru
- Trwały zapis danych

## Instrukcja uruchomienia
1. Otworzyć projekt w Android Studio
2. Uruchomić aplikację na emulatorze lub telefonie
3. Nadać wymagane uprawnienia (lokalizacja, aparat, aktywność)
4. Nacisnąć START, wykonać spacer i STOP + Zapisz
